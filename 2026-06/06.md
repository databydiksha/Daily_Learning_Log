# Day 2 - If-Else Statements

## What I learned

If-Else statements help a program make decisions based on conditions.

### Syntax

```python
if condition:
    # code
else:
    # code
```

## Example Code

```python
age = 20

if age >= 18:
    print("Eligible to vote")
else:
    print("Not eligible to vote")
```

## Key Takeaway

If-Else statements allow programs to perform different actions depending on whether a condition is True or False.
