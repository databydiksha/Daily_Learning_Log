# Day 10 - File Handling in Python

## What I learned

File handling allows Python programs to read from and write to files.

### Writing to a File

```python
file = open("sample.txt", "w")
file.write("Hello, Python!")
file.close()
```

### Reading from a File

```python
file = open("sample.txt", "r")
content = file.read()
print(content)
file.close()
```

## Why File Handling is Useful

- Store data permanently
- Read existing data
- Generate reports and logs

## Key Takeaway

Python can create, read, update, and manage files using built-in functions.

## Practice Example

```python
with open("notes.txt", "w") as file:
    file.write("This is my first file.")

with open("notes.txt", "r") as file:
    print(file.read())
```
