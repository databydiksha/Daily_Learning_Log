# Day 18 - Basic SQL Queries

## What I learned

SQL queries are used to retrieve and analyze data stored in a database.

### SELECT Statement

```sql
SELECT * FROM students;
```

### Selecting Specific Columns

```sql
SELECT name, marks
FROM students;
```

### Using WHERE Clause

```sql
SELECT *
FROM students
WHERE marks > 80;
```

## Why SQL Queries are Useful

- Retrieve required data
- Filter records efficiently
- Analyze information quickly

## Key Takeaway

The SELECT statement is the foundation of data retrieval in SQL.

## Practice Example

```sql
SELECT name
FROM students
WHERE marks >= 90;
```
