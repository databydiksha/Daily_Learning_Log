# Day 6 - Python Dictionaries

## What I learned

A dictionary stores data in key-value pairs.

### Creating a Dictionary

```python
student = {
    "name": "Diksha",
    "age": 21,
    "course": "MBA"
}

print(student)
```

### Accessing Values

```python
student = {
    "name": "Diksha",
    "age": 21
}

print(student["name"])
print(student["age"])
```

## Why Dictionaries are Useful

- Store related information together
- Fast access using keys
- Commonly used in data analysis and APIs

## Key Takeaway

Dictionaries store information as key-value pairs, making data organized and easy to access.

## Practice Example

```python
employee = {
    "name": "Rahul",
    "department": "Sales",
    "salary": 50000
}

print(employee["department"])
```
