# Day 5 - Python Lists

## What I learned

A list is used to store multiple items in a single variable.

### Creating a List

```python
fruits = ["apple", "banana", "mango"]
print(fruits)
```

### Accessing Elements

```python
fruits = ["apple", "banana", "mango"]

print(fruits[0])
print(fruits[1])
```

## Why Lists are Useful

- Store multiple values together
- Easy to update and modify
- Useful for loops and data handling

## Key Takeaway

Lists help organize and manage collections of data efficiently.

## Practice Example

```python
numbers = [10, 20, 30, 40]

numbers.append(50)

print(numbers)
```
