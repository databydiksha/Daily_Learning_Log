# Day 3 - Python Loops

## What I learned

Loops are used to execute a block of code multiple times.

### For Loop

A for loop is used when we know how many times we want to repeat something.

```python
for i in range(5):
    print(i)
```

### While Loop

A while loop runs as long as a condition is True.

```python
count = 1

while count <= 5:
    print(count)
    count += 1
```
## Key Takeaway

Loops help reduce repetitive code and make programs more efficient.

## Practice Example

```python
for number in range(1, 6):
    print("Number:", number)
```
