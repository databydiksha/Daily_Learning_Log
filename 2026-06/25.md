# Day 21 - SQL HAVING Clause

## What I learned

The HAVING clause is used to filter grouped data after applying the GROUP BY clause.

### HAVING Example

```sql
SELECT department, COUNT(*) AS Employee_Count
FROM employees
GROUP BY department
HAVING COUNT(*) > 5;
```

### HAVING with AVG()

```sql
SELECT department, AVG(salary) AS Average_Salary
FROM employees
GROUP BY department
HAVING AVG(salary) > 50000;
```

## Why HAVING is Useful

- Filters grouped records
- Works with aggregate functions
- Helps generate meaningful reports

## Key Takeaway

WHERE filters rows before grouping, while HAVING filters groups after grouping.

## Practice Example

```sql
SELECT city, COUNT(*) AS Customer_Count
FROM customers
GROUP BY city
HAVING COUNT(*) >= 10;
```
